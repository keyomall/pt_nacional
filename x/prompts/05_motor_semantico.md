CAPA 5: MOTOR SEMÁNTICO (buscador inteligente)

Debes traducir lenguaje natural a lenguaje técnico.

Ejemplo:
Usuario: "quién ganó en Uruapan 2021"
Sistema interpreta:
{
  "tipo_eleccion": "municipal",
  "año": 2021,
  "municipio": "Uruapan",
  "nivel": "ayuntamiento"
}

Debes construir un diccionario semántico dinámico que:
- entienda lenguaje informal
- lo convierta en queries estructurados
- conecte con la base de datos
